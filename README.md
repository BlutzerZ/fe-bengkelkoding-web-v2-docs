# Bengkel Koding Web V2 - Wiki Documentation

Selamat datang di dokumentasi resmi **Bengkel Koding Web V2**!

## Tentang Proyek

**Bengkel Koding** adalah platform pembelajaran pemrograman berbasis web untuk Universitas Dian Nuswantoro Semarang. Sistem ini menyediakan manajemen kursus, learning path, kelas, penugasan, presensi, dan sertifikat digital untuk mendukung proses belajar mengajar yang efektif.

### Tujuan Proyek

- Menyediakan platform pembelajaran pemrograman yang interaktif dan terstruktur
- Memfasilitasi manajemen kursus, kelas, dan penugasan secara digital
- Menyediakan sistem tracking progress mahasiswa
- Mengintegrasikan sistem presensi dan sertifikasi digital
- Mendukung multiple roles: Student, Asisten, Dosen, Admin, dan Super Admin

### Tech Stack

- **Framework**: Next.js 14.2.3 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **State Management**: Jotai
- **HTTP Client**: Axios
- **Authentication**: JWT (Cookie-based)
- **Editor**: Milkdown, React Markdown Editor
- **Charts**: ApexCharts
- **Package Manager**: Bun
